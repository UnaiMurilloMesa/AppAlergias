# Objetivo de esta app
## Esta aplicación móvil permite gestionar de forma sencilla la información de alumnos y sus alergias alimentarias dentro de un comedor escolar. 

### Su objetivo es optimizar y agilizar la identificación de alumnos alérgicos a la hora de servir las comidas, manteniendo una lista en tu teléfono de alumnos, alergias y platos con sus alérgenos.

### Es una herramienta práctica y útil para organizar el servicio diario y asegurar un control rápido y efectivo de los alérgicos sin necesidad de conexión a internet.

## CUIDADO⚠️ --> Esta aplicación utiliza el almacenamiento interno en el móvil. Si se desinstala o borra la caché, se eliminarán todos los datos introducidos relacionados a alumnos, alergias y comidas.

# Forma de uso
## 1 _ Gestión de cursos:
### Se deberá introducir todos los datos relacionados a los cursos y alérgeno que existan en el colegio correspondiente. Los datos pueden ser personalizados para adaptarlo a cualquier colegio o reglas, pero hay ejemplos con los cursos y alergias más comunes.
## 2 _ Inicialización de datos:
### Se deberá introducir los datos de todos los alumnos con los alérgenos que tienen. Esta app funciona en local, sin necesidad de acceso a internet, algo que es necesario para mantener la privacidad de todos los alumnos.
## 3 _ Creación de comidas y búsqueda de alérgicos:
### Al crear una comida será necesario introducir las alergias que contiene. Una vez añadida la comida, en la seccion de buscar alérgicos se podrá buscar ese plato para encontrar una lista con todos los alumnos alérgicos a alguno de los alimentos que contiene la comida.
